Eli Cohen | CS 35 Harvey Mudd | 5 February 2017

These tables summarize textual analysis of selected presidential inaugural addresses.
The Python scripts that support the analysis are part of the accompanying repository.
It tracks the instances of four words: "war","fear","hope","change".
